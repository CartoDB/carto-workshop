# Cartography with CARTO

## Agenda

* Cartographic design principles
* CartoCSS foundations
* CartoCSS basics:
  * Thematic mapping
  * Zoom based symbology
  * Torque: maps with temporal data
* Introduction to TurboCARTO
* Exercises with TurboCARTO
